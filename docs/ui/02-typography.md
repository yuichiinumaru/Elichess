# Typography

## Font Families

### Primary Interface
- **Variable:** `--font-main`
- **Value:** `'Inter', system-ui, sans-serif`
- **Usage:** Game controls, buttons, status text, instructions.

### Headings
- **Variable:** `--font-serif`
- **Value:** `'Georgia', serif`
- **Usage:** Main title ("Chess Server"), Modal headers. Adds a classic chess feel.

## Text Hierarchy

### H1 (Title)
- Font: Serif
- Color: Accent (Gold)
- Margin: Bottom spacing

### Status Text
- Size: `1.2rem`
- Dynamic Color:
    - Neutral: Text Color
    - Alert: Highlight Check (Red)

### Instructions
- Standard list styling within Cards.
- Font: Sans-serif
